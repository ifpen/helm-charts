MediaWiki v{{ .Chart.AppVersion }} a été déployé avec succès !

Pour accéder à MediaWiki :

{{- if .Values.ingress.enabled }}
1. Via Ingress :
{{- range $host := .Values.ingress.hosts }}
  http{{ if $.Values.ingress.tls }}s{{ end }}://{{ $host.host }}
{{- end }}
{{- else if contains "NodePort" .Values.service.type }}
1. Obtenez l'URL de l'application en exécutant :
  export NODE_PORT=$(kubectl get --namespace {{ .Release.Namespace }} -o jsonpath="{.spec.ports[0].nodePort}" services {{ include "mediawiki.fullname" . }})
  export NODE_IP=$(kubectl get nodes --namespace {{ .Release.Namespace }} -o jsonpath="{.items[0].status.addresses[0].address}")
  echo http://$NODE_IP:$NODE_PORT
{{- else if contains "LoadBalancer" .Values.service.type }}
1. Obtenez l'URL de l'application en exécutant :
     NOTE: La création d'un LoadBalancer peut prendre quelques minutes
  export SERVICE_IP=$(kubectl get svc --namespace {{ .Release.Namespace }} {{ include "mediawiki.fullname" . }} --template "{{"{{ range (index .status.loadBalancer.ingress 0) }}{{.}}{{ end }}"}}")
  echo http://$SERVICE_IP:{{ .Values.service.port }}
{{- else if contains "ClusterIP" .Values.service.type }}
1. Exécutez ces commandes pour accéder via port-forward :
  export POD_NAME=$(kubectl get pods --namespace {{ .Release.Namespace }} -l "app.kubernetes.io/name={{ include "mediawiki.name" . }},app.kubernetes.io/instance={{ .Release.Name }}" -o jsonpath="{.items[0].metadata.name}")
  echo "Visitez http://127.0.0.1:8080 pour utiliser votre wiki"
  kubectl --namespace {{ .Release.Namespace }} port-forward $POD_NAME 8080:80
{{- end }}

IMPORTANT : Lors du premier déploiement, le conteneur d'initialisation configure automatiquement la base de données MediaWiki.
Vérifiez les logs du conteneur init-db si le démarrage prend plus longtemps que prévu :
  kubectl logs --namespace {{ .Release.Namespace }} -l "app.kubernetes.io/name={{ include "mediawiki.name" . }},app.kubernetes.io/instance={{ .Release.Name }}" -c init-db

Si vous souhaitez réinitialiser manuellement la base de données ou mettre à jour le schéma :
  # Récupérer les mots de passe depuis les secrets Kubernetes
  export MW_DB_PASS=$(kubectl get secret --namespace {{ .Release.Namespace }} {{ include "mediawiki.fullname" . }} -o jsonpath="{.data.mariadb-password}" | base64 -d)
  export MW_ADMIN_PASS=$(kubectl get secret --namespace {{ .Release.Namespace }} {{ include "mediawiki.fullname" . }} -o jsonpath="{.data.mediawiki-admin-password}" | base64 -d)

  # Mise à jour du schéma (après une mise à jour de MediaWiki)
  kubectl exec --namespace {{ .Release.Namespace }} -it deployment/{{ include "mediawiki.fullname" . }} -- php /var/www/html/maintenance/update.php --quick
